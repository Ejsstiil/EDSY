About

This is a preview of EDSY's new design and layout. It is not yet complete! Some features (such as stored modules and API/Journal imports) have not yet been re-implemented, and there is plenty of polish and fine-tuning still to do. Please give it a spin and report any problems or suggestions using the contact links below.

Note that although stored builds are now supported on this preview site, they are still being kept separate from your live builds from the old design of the site, to avoid the chance of corrupting those builds in case there are bugs. Whenever you access this preview all of your live builds will be copied over for you to experiment with, but any changes made on this preview site will not affect the live site, and deleted builds will be restored from the live site the next time you return.

Options
— When enabled, the SLOT view of the module browser will display only the largest size class of each available module type.
— When enabled, the SLOT view of the module browser will display and allow you to fit modules that are too large for the current slot, or are of a type not allowed by the ship or the slot. This option can be used to experiment with theoretical "what-if" builds, but disallowed modules will be shown in red to signify that it will not be possible to actually fit them in-game.
